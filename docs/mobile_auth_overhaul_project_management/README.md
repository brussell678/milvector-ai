# Mobile Auth Overhaul Project Management Pack

This folder contains the project management control documents for the MilVector AI mobile-first UI overhaul and email/password authentication initiative.

Source PRD:

- `../milvector_mobile_auth_overhaul_prd.md`

Execution model:

- Work one sprint at a time.
- Keep changes scoped to the active sprint.
- Preserve existing business logic unless the active sprint explicitly requires a behavior change.
- Run `npm run build` before closing each sprint.
- Record remaining issues in the active sprint notes before moving forward.

Primary sprint sequence:

1. Sprint 1: Audit and Foundation
2. Sprint 2: Authentication Upgrade
3. Sprint 3: Mobile Navigation
4. Sprint 4: Dashboard Mobile Overhaul
5. Sprint 5: Transition Timeline Mobile Overhaul
6. Sprint 6: Tools Mobile Overhaul
7. Sprint 7: Library and Knowledge Base Mobile Overhaul
8. Sprint 8: Forms and Upload Warnings
9. Sprint 9: QA and Hardening
