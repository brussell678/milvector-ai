# Sprint 06 Context: Tools Mobile Overhaul

## Read First

- `../../Sprint Roadmap/README.md`

## Sprint Goal

Make AI tools usable on phones.

## Required Mobile Flow

1. Input
2. Generate
3. Review Output
4. Copy / Save / Download

## Allowed Work

- Refactor tool page layouts.
- Make forms single-column on mobile.
- Improve result panels.
- Add copy/download controls where practical.
- Add collapsible guidance for long instructions.

## Out of Scope

- Prompt redesign
- API route behavior changes
- Resume artifact schema changes unless fixing a bug uncovered by UI work

## Verification

- Run `npm run build`.
- Smoke test each tool page visually at mobile width.
