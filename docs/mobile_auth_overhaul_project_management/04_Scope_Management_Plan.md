# 04 Scope Management Plan

## In Scope

- Mobile-first responsive layout foundations
- Email/password signup and login
- Magic link preservation
- Password reset
- Mobile navigation
- Dashboard mobile layout
- Timeline mobile layout
- AI tool mobile usability
- Library and knowledge base mobile usability
- Upload warning standardization
- QA hardening

## Out of Scope

- Native mobile app
- Passkeys or biometric auth
- SMS authentication
- Offline-first mode
- Recruiter portal
- SkillBridge matching engine
- Full analytics dashboard redesign

## Scope Guardrails

- Work one sprint at a time.
- Do not refactor unrelated files.
- Do not remove existing routes.
- Do not make destructive database migrations.
- Keep existing product logic intact unless the active sprint requires a change.
