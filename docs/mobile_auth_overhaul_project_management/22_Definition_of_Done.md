# 22 Definition of Done

## Sprint Definition of Done

A sprint is done when:

- The active sprint scope is complete or explicitly deferred.
- `npm run build` has been run.
- Any build failure is documented with next action.
- Changed files are summarized.
- User-visible behavior is summarized.
- Acceptance criteria are checked.
- No unrelated refactor is included.

## Release Definition of Done

The release is done when:

- MilVector works well on mobile.
- Email/password authentication works.
- Magic link authentication remains available.
- Dashboard is usable on phones.
- Timeline is usable on phones.
- AI tools are usable on phones.
- Library and knowledge base are mobile-friendly.
- Upload warnings are present everywhere required.
- Production deployment succeeds.
