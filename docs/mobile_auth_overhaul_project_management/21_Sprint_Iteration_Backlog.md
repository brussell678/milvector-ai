## Handoff Files

Each sprint has two continuity files:

- `Sprints/Context/Sprint_XX_Context.md`: read before starting the sprint.
- `Sprints/Outbriefs/Sprint_XX_Outbrief.md`: complete before closing the sprint.

The next sprint should ingest all previous completed outbriefs plus its own context file.
