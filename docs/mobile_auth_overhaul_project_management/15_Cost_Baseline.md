# 15 Cost Baseline

## Baseline

No new vendor costs are planned. Work should use the existing stack and current subscription/services.

## Cost-Sensitive Areas

- OpenAI API usage
- Supabase storage
- Supabase auth email limits
- Vercel builds and hosting

## Control Measures

- Avoid new model calls during UI-only sprints.
- Avoid new dependencies unless justified.
- Reuse existing Supabase and OpenAI client patterns.
