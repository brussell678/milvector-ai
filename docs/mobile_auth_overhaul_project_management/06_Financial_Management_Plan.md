# 06 Financial Management Plan

## Purpose

Track cost-sensitive areas during the overhaul.

## Cost Drivers

- OpenAI API usage from integrated tools
- Supabase storage and database usage
- Vercel build and hosting usage
- Potential support time from auth or mobile regressions

## Cost Controls

- Preserve existing AI tool logic unless Sprint 6 requires UI-only changes.
- Avoid unnecessary client-side bundle growth.
- Keep magic link available but encourage email/password for restricted network reliability.
- Do not add new paid infrastructure without approval.

## Cost Review Points

- Sprint 2: auth behavior and support impact
- Sprint 6: AI tool usability without increasing model usage
- Sprint 9: deployment and runtime validation
