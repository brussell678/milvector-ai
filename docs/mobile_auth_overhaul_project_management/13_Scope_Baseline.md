# 13 Scope Baseline

## Product Scope

Deliver a mobile-first MilVector AI web application with reliable email/password authentication while preserving existing magic-link authentication and current product workflows.

## Deliverables

- Responsive app shell foundations
- `/auth` email/password and magic-link experience
- Mobile navigation
- Mobile dashboard
- Mobile timeline
- Mobile tool pages
- Mobile library and knowledge base
- Standard upload warning
- QA hardening

## Exclusions

- Native app
- SMS auth
- Passkeys
- Full admin mobile redesign
- New product modules outside the PRD
